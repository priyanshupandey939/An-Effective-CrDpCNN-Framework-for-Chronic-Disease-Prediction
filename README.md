Early and accurate prediction of these diseases is crucial for enhancing patient outcomes and minimizing overall healthcare cost. This study presents an efficient deep learning framework known as the Chronic Disease Prediction Convolutional Neural Network (CrDpCNN) for accurate prediction using structured healthcare data. Firstly, the healthcare data is pre-processed using normalization, correlation-based feature selection, and dimensionality reduction techniques to improve the quality and reduce the redundancy of the data. Then, the refined feature set is used to train the proposed CrDpCNN framework to effectively learn complex patterns in healthcare data. Finally, the performance of the proposed framework is validated using k-fold validation to ensure the efficiency of the framework. The results show that the CrDpCNN framework has the capability to attain accuracy up to 99.18% using the 80-20 split ratio. Moreover, to validate the robustness and reliability of the CrDpCNN framework, the model was validated using 5-Fold Validation, and the accuracy was found to be 98.25%. Moreover, the framework is also good at precision, recall, and F1-score.
